## 附录：如何打包 Android APK
1. 前置检查（先看 Android 工具链是否可用）：
```bash
flutter doctor -v
```
要求 `Android toolchain` 为 ✅。

2. 如果提示找不到 SDK（`No Android SDK found`），先配置：
```bash
flutter config --android-sdk ~/Library/Android/sdk
flutter doctor --android-licenses
flutter doctor -v
```

3. 构建 release APK：
```bash
cd /Users/jyy/Documents/bihua
flutter build apk --release
```

4. 产物路径：
- `/Users/jyy/Documents/bihua/build/app/outputs/flutter-apk/app-release.apk`

5. 验证 APK 签名信息（可选）：
```bash
~/Library/Android/sdk/build-tools/36.0.0/apksigner verify --print-certs /Users/jyy/Documents/bihua/build/app/outputs/flutter-apk/app-release.apk
```

6. 安装到 Android 设备（USB 调试）：
```bash
adb devices
adb install -r /Users/jyy/Documents/bihua/build/app/outputs/flutter-apk/app-release.apk
```

7. 说明：
- 当前构建出的 `app-release.apk` 为调试证书签名，适合测试安装。
- 如果需要对外分发和后续版本覆盖升级，需改为正式 keystore 签名。
